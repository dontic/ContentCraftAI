## Get started
### Self Hosted

You need to have API access to:
- OpenAI or Claude
- SerpAPI

You can then deploy the tool locally with docker:
  
```bash
docker run -p 8501:8501 dontic/postifyai:latest
```

The tool will be available at `http://localhost:8501`
